# bourboneat
Wordpress Theme (v4.1 and later) with SASS, Bourbon.io Neat and IcoMoon.io icons

## setup
<pre><code>
$ npm install
$ bower install
</code></pre>

gulp tasks
---------
<pre><code>
$ gulp styles // recompile sass
$ gulp js // recompile js
</code></pre>
